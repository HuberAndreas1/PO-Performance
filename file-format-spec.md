# CyberLift Log Specification

## 1. Background

A CyberLift Log File is a structured data format used to exchange workout data from the SmartPass hardware to the web portal. It uses a state-based structure where a session date is defined once, followed by multiple exercises and their corresponding sets.

The "Sync Station" must process raw text files (`gym_logs.txt`) generated by the SmartPass.

---

## 2. Format Example
```text
### 2026-02-06
Chest Press

1. 85kg x 10
2. 80kg x 9
Preacher Curls

1. 11.25kg x 8
2. 8.75kg x 9 | easy
````

---

## 3. Structure Rules

| Element | Rule |
|---------|------|
| **Session Header** | Lines starting with `###` followed by a date in `YYYY-MM-DD` format. |
| **Exercise Name** | A plain text line following a header or a set list. |
| **Set Entry** | Format: `[Index]. [Weight]kg x [Reps]` |
| **Commentary** | Optional, separated by a pipe `\|` |
| **Whitespace** | Ignore empty lines between exercises/sets. |

---

## 4. File Validation & Error Handling

Your parser **must validate every line** and throw the appropriate error if the format is violated.

### 4.1 Error Definitions

| Error Code | Description |
|------------|-------------|
| `EmptyFile` | The file has no content. |
| `MissingSessionHeader` | The first non-empty line does not start with `###`. |
| `InvalidSessionDateFormat` | The `###` date is malformed or invalid. |
| `FutureDateError` | The session date is in the future. |
| `MultipleSessionsInFile` | A second `###` header is found. Only one session per file is allowed. |
| `ExerciseNameTooShort` | Exercise name is **< 3** characters. |
| `ExerciseNameTooLong` | Exercise name is **> 100** characters. |
| `DuplicateExerciseInSession` | The same exercise name appears twice within the file. |
| `MissingExerciseName` | A set (`1. ...`) appears before an exercise is defined. |
| `EmptyExercise` | An exercise name is defined, but no sets follow before the next exercise begins or the file ends. |
| `InvalidSetFormat` | Set line is missing the `.` or the `x` separator. |
| `InvalidWeightFormat` | Weight is missing `kg` or isn't a valid number. |
| `InvalidRepsFormat` | Repetition count is not a valid number. |
| `InvalidSetSequence` | Set index is not exactly 1 higher than the previous (e.g., `1, 2, 4`). |
| `NegativeValueDetected` | Weight or Reps are **≤ 0**. |
| `CommentaryTooLong` | Text after the `\|` exceeds **250 characters**. |
